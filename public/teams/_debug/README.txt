Источник правды для иконок команд.

1. Кладите сюда JPG/PNG с именем команды: amur.jpg, cska.jpg, ska.jpg …
2. Импорт в приложение: npm.cmd run teams:import
3. НЕ запускайте teams:slice — он только черновик в _slice_preview и часто путает порядок.

Файлы вида 00_amur.jpg сюда не класть — они не используются.
