# Go Hockey — спецификация (черновик)

Приложение для своих: один город, доступ через администратора.

## Роли

| Роль | Описание |
|------|----------|
| **admin** | Создаёт логины, запускает голосования и оплату, управляет составом |
| **player** | Участник по умолчанию после активации |

## Группы

- Группы организованы **по дням** (игровой день / слот).
- В группе — список участников с признаком `actual` (активен в этой группе).

## Поток пользователя

### 1. Выдача доступа (админ)

1. Админ создаёт учётную запись: логин = **номер телефона**, генерируется **временный пароль**.
2. Админ отправляет игроку логин и пароль (вне приложения).

### 2. Первый вход (игрок)

1. Авторизация по телефону + паролю.
2. Обязательно: **сменить пароль** и задать **свой логин** (отображаемое имя).
3. После этого игрок **активен**; в текущей группе дня — `actual = true`.

### 3. Голосование (админ → игроки)

1. Админ **запускает голосование** в группе дня.
2. Всем участникам — **уведомление**.
3. Голосование в группе **активно** ограниченное время.
4. По времени принятия голосов формируется **состав**:
   - первые **20** — в игру;
   - остальные — **резерв** (по порядку времени голоса).

### 4. Управление составом (админ)

- Удалить из проголосовавших может **только админ**.
- При удалении — признак **не участвует**; список пересчитывается (очередь/резерв).

### 5. Оплата

1. Админ запускает этап **уведомлений об оплате**.
2. Игрок отмечает **«оплатил»**.
3. Админ видит статусы; для неоплативших — **повторная рассылка** уведомления.

### 6. Гости и очередь (админ)

- Админ может **добавлять гостей** (без обычной регистрации).
- Админ может **менять очередь** и **вставлять без очереди**.

## Инфраструктура (решение)

- **Хостинг reg.ru:** статика PWA (`dist/`) + API на **PHP + MySQL** (типично для shared-хостинга).
- **Масштаб:** до ~150 пользователей — MySQL на shared более чем достаточно.
- **Push:** Web Push (HTTPS + service worker + подписки в БД + отправка с сервера).
- **Админ:** только с телефона — достаточно адаптивного веб-интерфейса.
