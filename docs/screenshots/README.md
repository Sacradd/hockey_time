# Скриншоты для инструкции

Папка для PNG/JPG к [USER-GUIDE.md](../USER-GUIDE.md).

## Как снять

1. iPhone: Safari или PWA с иконки (лучше для prod-вида).
2. Или эмулятор / dev (`npm.cmd run dev:lan`) — для черновика.
3. Имена файлов — как в чеклисте в `USER-GUIDE.md` (внизу документа).

## Рекомендации

- Одна тема на файл, без лишнего UI ОС.
- Ширина ~390px (iPhone) достаточно; можно WebP для веса.
- Не коммитьте скрины с **реальными телефонами** игроков без согласия — размывайте или тестовые аккаунты.
