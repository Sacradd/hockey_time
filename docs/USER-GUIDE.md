# Инструкция по приложению «Время хоккея»

> Черновик для игроков и админов группы.  
> **Скриншоты:** положите файлы в `docs/screenshots/` (см. [screenshots/README.md](./screenshots/README.md)) и подставьте пути в разделы ниже.

## Установка на iPhone (PWA)

1. Откройте сайт в **Safari** (после деплоя на HTTPS).
2. «Поделиться» → **«На экран "Домой"»**.
3. Запускайте **с иконки**, не из закладки — тогда полноэкранный режим без панелей Safari.

![Установка PWA](screenshots/pwa-install.png)

## Общее для всех

### Вход

- Логин и временный пароль выдаёт **администратор группы**.
- При первом входе: смена пароля и выбор **ника** (отображаемого имени).

![Экран входа](screenshots/login.png)

### Верхняя панель

- Слева: **профиль** (аватар, ник).
- Справа: **«Назад»** (где есть куда вернуться) и **выключение** (выход из аккаунта).

![Верхняя панель](screenshots/top-bar.png)

### Профиль

- Ник, любимая команда КХЛ, список **«Мои группы»**.
- В группе: участники (полевые / вратари / админы), кнопка «Управление группой» — только у админа.

![Профиль и мои группы](screenshots/profile-rosters.png)

## Роль: игрок

### Главная — блок «Игры»

- Карточки предстоящих игр и игр с активным голосованием / оплатой / опубликованными составами.
- Бейджи: **голосование**, **оплата**, **Составы сформированы** (зелёный).

![Список игр на главной](screenshots/player-home-games.png)

### Страница игры (до публикации составов)

| Ситуация | Что видно |
|----------|-----------|
| Голосование не запущено | Текст «Голосование не запущено» |
| Голосование открыто | Кнопки **«Буду»** / **«Не буду»** (и третий вариант, если задал админ) |
| Вас включили в «будут» | Статус вверху (например, в основе / резерв) |
| Оплата | Кнопка подтверждения оплаты (только полевые в основе) |

- Обновить данные: **потянуть список вниз** вверху экрана (pull-to-refresh).

![Голосование](screenshots/player-vote.png)

### После публикации составов

- Только блок **белые / чёрные** и строка **«Вы в команде …»** (белые — белый текст, чёрные — оранжевый).
- Голосование и оплата на этом экране скрыты.

![Опубликованные составы](screenshots/player-teams-published.png)

## Роль: администратор группы

### Главная

- Блок **«Группы»** — управление roster (создание группы — у super).
- Блок **«Игры»** — свайп **влево** по карточке:
  - до публикации составов → **«Удалить»** (игра стирается);
  - после **«Готово»** по составам → **«В архив»** (пропадает из списка, данные сохраняются).

![Свайп по игре](screenshots/admin-game-swipe.png)

### Группа (roster)

- Игры на даты, участники (свайп влево — удалить из группы, не себя).
- Создание игры на дату.

![Страница группы](screenshots/admin-roster.png)

### Страница игры

1. **Запустить голосование** — подписи ответов, срок.
2. **Требование об оплате** — параллельно с голосованием; вратари не платят.
3. Состав: свайп «выбыл», ₽ у полевых, гости, ручная очередь.
4. **«Сформировать состав»** → белые/чёрные → **«Готово»** — все видят только составы.
5. Редактирование игры (карандаш): дата/время; удалить или **в архив** (если составы уже опубликованы).

![Админ: управление игрой](screenshots/admin-game.png)

![Формирование составов](screenshots/admin-teams.png)

| Сформировать белые/чёрные | — | ✅ |
| Удалить игру | — | ✅ до «Готово» |

| Удалить игрока из группы | — | ✅ свайп в roster |
